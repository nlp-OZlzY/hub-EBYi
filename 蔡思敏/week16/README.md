# AI 狼人杀 — 多智能体协作与博弈的 Agent Team 实战

## 项目简介
基于多 Agent 协作框架，构建一个能够自主完成信息不对称博弈的狼人杀 Agent Team 系统。课题核心在于多智能体的协作/对抗与交互机制设计：每个 Agent 根据其扮演角色（狼人、预言家、女巫等）拥有独立的目标、策略与行动空间，在严格信息隔离的约束下进行推理、发言与决策。

## 核心特性
- **多智能体协作/对抗**: 每个 Agent 根据角色拥有独立目标、策略与行动空间
- **严格信息隔离**: 狼人、预言家、女巫等角色信息严格隔离
- **完整对局引擎**: 驱动回合流转与胜负裁决
- **结构化日志**: 全程可观测的对局日志，保存到 `logs/` 目录
- **前端观战 UI**: 支持纯 AI 对战或人机混战（可选）
- **自进化系统**: "对局→分析→优化→再对局"循环（进阶方向③）

## 快速开始

### 方式一：简化版（推荐，无需安装依赖）
```bash
python werewolf.py
```

### 方式二：模块化版本
```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python src/main.py

# 自进化训练
python src/evolution.py
```

## 角色说明
| 角色 | 阵营 | 特殊能力 |
|------|------|----------|
| 狼人 (3人) | 狼人阵营 | 夜间杀人 |
| 预言家 (1人) | 好人阵营 | 夜间查验身份 |
| 女巫 (1人) | 好人阵营 | 解药/毒药 |
| 猎人 (1人) | 好人阵营 | 死亡时开枪 |
| 守卫 (1人) | 好人阵营 | 夜间守护 |
| 平民 (若干) | 好人阵营 | 无特殊能力 |

## 游戏流程
1. **夜晚阶段**
   - 狼人行动：选择攻击目标
   - 守卫行动：选择守护目标
   - 预言家行动：查验一名玩家身份
   - 女巫行动：使用解药或毒药

2. **白天阶段**
   - 宣布死亡结果
   - 讨论阶段：所有存活玩家发言
   - 投票阶段：投票选出可疑玩家
   - 宣布投票结果

3. **胜负判定**
   - 好人胜利：所有狼人被淘汰
   - 狼人胜利：狼人数量 >= 好人数量

## 进阶功能

### ① 通用 Agent
探索"读懂自己→修改自己→运行自己"的自演化系统，从「通用 Agent」演化为「狼人杀多角色 Agent」。

### ② 评测+复盘
构建结果评测、过程评测等多维可量化评测体系，基于此完成任意游戏的复盘归因，并产出不同版本、不同模型 Agent 同台竞技的 Leaderboard。

### ③ 自进化 Agent
实现"对局→分析→优化→再对局"自进化循环，使各角色 Agent 在多局迭代中持续提升胜率。

```bash
python src/evolution.py
```
