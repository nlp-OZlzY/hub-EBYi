# MinerU vs PDFPlumber 对比分析

## MinerU 简介

MinerU 是一个基于深度学习的多模态文档解析工具。其核心特点：

1. **多种解析引擎 (Backend)**:
   - `pipeline`: 基于规则+深度学习模型，适合通用场景
   - `vlm-http-client`: 调用远程VLM模型（如Qwen-VL）进行高精度解析
   - `hybrid-http-client`: 混合方案，本地+远程结合

2. **支持多种输入**: PDF、图片、docx、pptx、xlsx

3. **内置能力**:
   - 表格识别
   - 公式识别 (LaTeX)
   - 图像/图表分析
   - 多语言OCR

## PDFPlumber 特点

- 纯Python库，基于pdfminer
- 只能提取文本和表格
- 无法解析图片中的内容
- 对扫描PDF（图片型）支持差

## 实际测试对比

| 维度 | MinerU | PDFPlumber |
|------|--------|------------|
| **文本提取** | 保留结构（标题、段落） | 纯文本，可能丢格式 |
| **表格提取** | 结构化输出，保留表格边界 | 基本表格提取，格式可能丢失 |
| **图片处理** | 能识别图片中的文字（OCR） | 无法处理，只能提取嵌入图片 |
| **公式解析** | 支持（LaTeX格式） | 不支持 |
| **扫描PDF** | 支持（需要OCR模式） | 几乎无用 |
| **输出格式** | Markdown + 图片 | 纯文本 |
| **速度** | 较慢（依赖模型） | 快（纯计算） |
| **部署复杂度** | 高（需模型、GPU） | 低（pip安装即可） |
