## 回溯搜索

·每次扩展只生成一个子节点，而不是所有子节点

空间复杂度

空间复杂度： $O ( b d )$

## 双向搜索

·两个搜索同时运行：一个从初始状态向前搜索，另一个从目标状态向后搜索，直到状态相遇

目标测试：检查两个搜索的边界集是否相交

优点：搜索区域比深度优先搜索小

·限制：只能用在支持反向搜索的场合

最优性：非最优

·双向宽度优先搜索的时间复杂度： $O ( b ^ { d / 2 } )$

·双向宽度优先搜索的空间复杂度： $O ( b ^ { d / 2 } )$
