## 可容许性：

函数永远不会高估到达某个目标的代价 $\mathbf { h } ( \mathbf { n } ) \leq \mathbf { h } ^ { * } ( \mathbf { n } )$

·f(n)不会超过经过节点n的最优解的实际代价

·若h(n)是可容许的，则A搜索被称为A\*搜索，具有最优性

授课教师：李静瑶

## A\*搜索

## 可容许性：

反证法证明最优性。假设最优路径的代价为 $\mathbf { \nabla } \mathbf { C } ^ { * }$ ，但返回 $\ b { \mathrm { C } } > \ b { \mathrm { C } } ^ { * }$ ，那么最优路径上一定存在节点n未扩展。 $g ^ { * } ( n )$ 表示从起点到n的最优路径代价， $h ^ { * } ( n )$ 表示从n到最近目标的最优路径的代价。

<table><tr><td rowspan=1 colspan=1> $f ( n ) > C ^ { * }$ </td><td rowspan=1 colspan=1>否则n将是已扩展节点</td></tr><tr><td rowspan=1 colspan=1> $f ( n ) = g ( n ) + h ( n )$ </td><td rowspan=1 colspan=1>根据定义</td></tr><tr><td rowspan=1 colspan=1> $f ( n ) { = } g ^ { * } ( n ) + h ( n )$ </td><td rowspan=1 colspan=1>因为n在一条最优路径上</td></tr><tr><td rowspan=1 colspan=1> $f ( n ) \leq g ^ { * } ( n ) + h ^ { * } \left( n \right)$ </td><td rowspan=1 colspan=1>根据可容许行， $h ( n ) \leq h ^ { * } ( n )$ </td></tr><tr><td rowspan=1 colspan=1> $f ( n ) \leq C ^ { * }$ </td><td rowspan=1 colspan=1>根据定义，  $C ^ { * } { = } g ^ { * } \left( n \right) { + } h ^ { * } \left( n \right)$ </td></tr></table>

第一行与最后一行矛盾，“算法返回次优路径"的假设是错误的。

## 一致性:

·对于任意节点n和通过动作a生成的后继结点n'，从n到达目标的估计代价不大于从n到n'的单步代价与从n到达目标的估计代价之和： $\mathrm { h ( n ) } \leq$ $\operatorname { c } ( \mathrm { n , a , n ^ { \prime } } ) + \mathrm { h } ( \mathrm { n ^ { \prime } } )$

## 启发式函数的准确性

·描述启发式函数质量的方法：有效分支因子b\*

·若A\*算法的解路径长度为d，生成结点数为N，则b\*是深度为d的均衡搜索 树生成N+1个节点的有效分支因子，满足 $1 + \mathrm { b } ^ { * } + \mathrm { b } ^ { * 2 } + \ldots + \mathrm { b } ^ { * \mathrm { d } } = \mathrm { N } + 1$

·b\*越小越好（N越小越好），最好情况为1

目标：设计启发式函数，使得b\*尽可能的小
