# 股票波动可视化与买卖时机分析

## 你需要做什么

当用户请求分析某只股票时，按以下流程执行：

### 1. 确认参数

从用户消息中提取：
- **股票代码**（必需）：6 位数字，如 `000001`
- **股票名称**（可选）：如 `平安银行`，未提供时用代码代替
- **时间范围**（可选）：默认最近 90 天

如果用户没有提供股票代码，先询问。

### 2. 执行分析脚本

运行捆绑的 `scripts/stock_viz.py`：

```bash
python scripts/stock_viz.py --code <股票代码> --name <股票名称> --days <天数>
```

这个脚本会自动完成：
- 从 autostock.cn API 获取日K线和周K线数据
- 计算每日/每周涨跌幅（波动率）
- 生成 `output/<代码>_volatility.png` 图表（上下子图：日波动柱状图 + 周波动柱状图，绿涨红跌）
- 在控制台输出分析报告

### 3. 解读结果向用户汇报

脚本输出包含两部分，你需要整合后向用户说明：

**图表**（`output/<代码>_volatility.png`）：直接展示给用户。

**分析报告**（控制台输出），包含：
- 基本指标：当前价格、近期日均波动、周均波动、今日/本周波动
- 趋势方向：日趋势和周趋势（向上/向下）
- 价格分位：当前价在 20 日区间的相对位置
- 信号强度：买入分数 vs 卖出分数
- 具体交易信号及理由

向用户汇报时，用通俗语言总结关键发现，不要只复述数字。例如：
> "平安银行目前日波动收缩明显，周趋势向上，价格处于中低位——技术面偏向买入。但买入信号强度一般，建议关注明日走势确认。"

### 4. 免责提醒

每次分析结束后附加一句：
> ⚠️ 以上分析仅基于技术面波动指标，不构成投资建议。实际交易请结合基本面和市场环境判断。

## 脚本依赖

首次使用前，确保安装依赖：

```bash
pip install -r requirements.txt
```

## 分析逻辑参考

以下逻辑已内置于 `scripts/stock_viz.py`，仅供参考理解：

| 条件 | 信号类型 | 强度 | 核心逻辑 |
|------|----------|------|----------|
| 日波动 < 均值-0.8σ 且周趋势向上 | 买入 | 强 | 波动收缩蓄力突破 |
| 连续3日波动递减 + 价格 < 60%分位 | 买入 | 中 | 变盘前兆 |
| 周波动 1-3% + 日趋势转正 + 价格 < 50%分位 | 买入 | 中 | 健康回调结束 |
| 日波动 > 均值+1.5σ 且价格 > 70%分位 | 卖出 | 强 | 异常波动见顶 |
| 周波动 > 均值×1.5 且日趋势转负 | 卖出 | 中 | 风险上升 |
| 连续3日大涨 + 波动急剧放大(>均值×2) | 卖出 | 强 | 短期过热 |

### 波动水平定性评估

报告还包含波动大小的定性判断：

| 日波动均值 | 定性 | 周波动均值 | 定性 |
|-----------|------|-----------|------|
| < 0.8% | 偏低，股价平稳 | < 2% | 偏低，中期趋势平稳 |
| 0.8%-1.5% | 适中，正常范围 | 2%-4% | 适中，正常范围 |
| 1.5%-2.5% | 偏高，波动明显 | 4%-7% | 偏高，中期波动较大 |
| > 2.5% | 很高，剧烈波动 | > 7% | 很高，中期剧烈波动 |

## 输出文件位置

- 图表：`output/<股票代码>_volatility.png`
- 脚本本身不持久化报告，报告内容在控制台输出中
