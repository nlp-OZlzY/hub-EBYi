###  阅读 2412-DeepSeek-V3.pdf 论文，总结 V3 相比 V2 改进
  DeepSeek-V3 相比 V2，我觉得本质上不是推翻重做，而是在 V2 已验证的高效架构上做了一次全面升级。
  首先，基础架构没变，还是沿用 V2 的 MLA 和 DeepSeekMoE，说明 V2 的路线是对的。但 V3 做了几个关键增强：第一，它在 MoE 路由里引入了无辅助损失的负载均衡策略，比传统依赖 auxiliary loss 的方法更能兼顾负载均衡和模型性能；第二，新增了 MTP，多 token 预测目标，不仅提升训练效果，还能支持 speculative decoding，加快推理；第三，预训练数据更强了，明显提高了数学、代码和多语言数据比例，而且训练 token 规模扩大到了 14.8T；第四，工程上升级很大，V3 用了 FP8 混合精度训练，还重点优化了跨节点 MoE 通信，所以训练效率和成本都更好；第五，后训练阶段又结合了 DeepSeek-R1 的推理蒸馏，所以 V3 的 reasoning 能力也比 V2 更强。
