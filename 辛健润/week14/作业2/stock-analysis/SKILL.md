# 股票分析 Skill

本 Skill 用于分析股票市场数据，包括价格走势、波动率分析、可视化图表生成和智能投资建议。

## 触发场景

以下场景应使用本 Skill:
- 分析特定股票的历史表现
- 计算股票的日波动率和周波动率
- 生成股票分析可视化图表
- 获取基于 AI 的买卖建议
- 识别最佳买入/卖出时机

## 核心功能

### 1. 数据获取
- 从 Yahoo Finance 获取实时股票数据
- 支持美股、A股、港股等多个市场
- 可自定义分析周期（1天到多年）

### 2. 波动率分析
- **日波动率**（5日滚动窗口）：反映短期价格波动
- **周波动率**（20日滚动窗口）：反映中期趋势
- 年化波动率计算
- 识别高/低波动期

### 3. 可视化图表
生成三合一专业图表：
- **价格走势图**：收盘价曲线 + 日内波动范围
- **成交量图**：红涨绿跌柱状图
- **波动率对比图**：日波动率 vs 周波动率

### 4. 交易信号分析
- 识别历史最佳买入时机（低波动 + 相对低价）
- 识别历史最佳卖出时机（高价格 + 适度波动）
- 计算潜在收益和收益率
- 统计低波动期和高波动期天数

### 5. AI 投资建议
使用大语言模型分析数据并提供：
- 当前是否适合买入的判断
- 建议的卖出时机
- 风险提示和注意事项
- 具体的操作策略建议

#### 2. 波动率计算
```python
def calculate_volatility(prices: pd.Series, window: int) -> pd.Series:
    """
    计算滚动波动率

    公式：
    1. 计算日收益率 = (今日价格 - 昨日价格) / 昨日价格
    2. 计算收益率的滚动标准差
    3. 年化处理 = 标准差 × √252
    """
    returns = prices.pct_change()
    volatility = returns.rolling(window=window).std() * np.sqrt(252)
    return volatility
```

   - 不构成任何投资建议
