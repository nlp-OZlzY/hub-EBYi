# 股票波动分析 Skill

~~~markdown
# name
股票波动分析

# description
分析股票的日波动与周波动，并根据波动极值提供买入/卖出建议，适用于股票走势可视化和投资参考。

# quick start
ticker: 600519
period: 3mo

# inputs
| 名称   | 类型   | 描述                                 | 默认值 |
|--------|--------|------------------------------------|--------|
| ticker | 字符串 | 股票代码，例如 `600519`（贵州茅台） | 600519 |
| period | 字符串 | 历史数据时间范围，例如 `3mo`、`6mo`、`1y` | 3mo |

# outputs
| 名称              | 类型    | 描述                                           |
|------------------|---------|-----------------------------------------------|
| plot_url          | 字符串  | 生成的日/周波动可视化图表 URL 或文件路径      |
| daily_change      | 数字    | 最新日波动值（今日收盘价 - 今日开盘价）       |
| weekly_change     | 数字    | 最新周波动值（本周收盘价 - 上周收盘价）      |
| action_suggestion | 字符串  | 基于波动极值的建议：`考虑买入`、`考虑卖出` 或 `持有` |

# examples
## 示例 1
**输入**
```text
ticker: 600519
period: 3mo
~~~
