# 股票 Skill 说明

## 一、stock-visual：股票波动可视化与买卖建议

### 设计目标
从周线和日线两个维度综合分析股票走势，将中长期趋势（周线）与短期波动（日线）绘制在同一张图中，让投资者一目了然地看到多空力量对比和买卖时机。

### 核心分析逻辑

**1. 均线系统（MA5 / MA20）**
- MA5（5周期均线）反映短期价格方向
- MA20（20周期均线）反映中期价格方向
- 两条均线的关系是判断趋势的核心依据

**2. 金叉死叉信号检测**
- 金叉：短期均线从下方向上穿越长期均线 → 买入信号（趋势由空转多）
- 死叉：短期均线从上方向下穿越长期均线 → 卖出信号（趋势由多转空）
- 分别在周线和日线上独立检测，周线信号权重更高

**3. 趋势判断**
- 上升趋势：收盘价 > MA5 > MA20，价格在均线上方运行
- 下降趋势：收盘价 < MA5 < MA20，价格在均线下方运行
- 震荡整理：价格在均线附近反复穿越

**4. 波动率计算**
- 使用收盘价的滚动标准差衡量波动幅度
- 高波动说明市场分歧大，可能出现趋势转折
- 低波动说明市场平静，可能蓄势待发

**5. 综合评分制买卖建议**
采用加权打分制决定最终建议，各项权重和逻辑如下：

- 周线趋势为上升 → +2分（周线定大方向，权重最高）
- 周线趋势为下降 → -2分
- 日线趋势为上升 → +1分（日线反映短期节奏，权重次之）
- 日线趋势为下降 → -1分
- 周线最近出现金叉 → +2分（周线信号更可靠）
- 周线最近出现死叉 → -2分
- 日线最近出现金叉 → +1分
- 日线最近出现死叉 → -1分

根据总分给出建议：
- ≥3分 → 强烈买入：周线多头，日线配合，信号一致
- 1~2分 → 逢低买入：偏多但不够强，等待回调入场
- 0分 → 观望：无明显方向
- -1~-2分 → 逢高卖出：偏空，反弹减仓
- ≤-3分 → 强烈卖出/观望：周线日线均空头

### 图表结构（3张子图）
1. **价格走势+均线**：周K收盘价 + 周MA5/MA20 + 日K收盘价叠加，标注金叉死叉
2. **波动率对比**：周波动率 + 日波动率双Y轴，标注高波动区间
3. **周成交量**：红涨绿跌，辅助判断信号强度

## 二、stock-volatility：股票周日波动对比与买卖建议

### 设计目标
这是对 stock-visual 的功能变体，复用了 autostock API 和整体架构。区别在于分析维度从"均线趋势"切换为"波动率变化"，通过波动率的扩张和收敛来判断市场情绪和转折点。

### 为什么这样设计（复用+差异化）

1. **复用数据接口**：get_week_kline / get_day_kline 函数与 stock-visual 完全一致，不重复造轮子
2. **差异化分析角度**：stock-visual 看的是"方向"（涨还是跌），这个 skill 看的是"情绪"（波动激烈还是平静）
3. **互为补充**：实际交易中，趋势+波动率需要结合使用——均线告诉你方向，波动率告诉你时机

### 核心分析逻辑

**1. 三种波动率指标**
- 绝对波动率 = 最高价 - 最低价 → 当日/当周的价格振幅
- 相对波动率 = (最高价 - 最低价) / 收盘价 × 100% → 消除价格量纲，不同股票可比较
- 滚动波动率 = 收盘价N日标准差 → 衡量价格的不确定性

**2. 波动率状态判断**
- 高波动：当前值 > 历史均值 × 1.2 → 市场分歧大，可能是顶/底
- 低波动：当前值 < 历史均值 × 0.8 → 市场平静，可能是暴风雨前的宁静
- 中波动：接近均值 → 正常运行状态

**3. 波动率趋势判断**
- 比较最近5期和前5期的均值
- 扩大（recent > prev × 1.15）→ 波动加剧，风险增大
- 收敛（recent < prev × 0.85）→ 波动缩小，蓄势待发
- 稳定 → 延续当前状态

**4. 买卖信号逻辑（波动率视角）**
- 低波动 + 收敛 + 价格温和上升 → 蓄势待发，买入时机（+3分）
- 低波动 + 收敛 → 可能有突破（+2分周线，+1分日线）
- 高波动 + 扩大 + 价格走弱 → 风险增大，卖出信号（-2分）
- 高波动 + 扩大 → 风险偏高（-1分）
- 价格趋势配合：近期价格上涨 → +1分，下跌 → -1分

**5. 综合建议（分数制，同 stock-visual）**
- ≥3分 → 强烈买入
- 1~2分 → 逢低买入
- 0分 → 观望
- -1~-2分 → 逢高卖出
- ≤-3分 → 卖出/观望

### 图表结构（3张子图）
1. **相对波动率对比**：周/日相对波动率双Y轴，标注均值线
2. **滚动波动率+信号区间**：高波动区（红色）和低波动蓄势区（绿色）标注
3. **周成交量**：配合波动率判断信号可靠性

## 三、两个 Skill 的对比总结

| 维度 | stock-visual | stock-volatility |
|------|-------------|-----------------|
| 分析视角 | 趋势方向（涨跌） | 市场情绪（波动大小） |
| 核心指标 | MA5/MA20均线、金叉死叉 | 绝对/相对/滚动波动率 |
| 买入信号 | 金叉 + 上升趋势 | 低波动收敛 + 价格企稳 |
| 卖出信号 | 死叉 + 下降趋势 | 高波动扩大 + 价格走弱 |
| 适合场景 | 趋势跟踪交易 | 波动率交易 / 突破交易 |
| 数据来源 | 同一 API（autostock.cn） | 同一 API，复用 stock-visual |
