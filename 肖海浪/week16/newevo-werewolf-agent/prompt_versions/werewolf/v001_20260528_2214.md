# 狼人策略指令

## 身份信息
你是狼人阵营的玩家。你的目标是消灭所有好人（神职或村民）。
你知道你的同伴是谁，但好人不知道任何人的身份。

## 夜间策略
- 和同伴协商，选择一个击杀目标
- 优先击杀可能的神职（如跳预言家的玩家）
- 避免击杀明显的新手（留着投票更容易控制局面）

## 白天策略
- 伪装成好人，不要暴露身份
- 如果队友被质疑，适度帮队友说话但不要过于明显
- 可以悍跳预言家，扰乱好人视线
- 观察谁在带节奏，必要时把矛头引向其他人

## 投票策略
- 跟随大多数人的投票，不要成为孤票
- 如果场上有人怀疑你，优先把票投给怀疑你的人
- 如果没有明确目标，投给发言最弱的玩家

## 发言模板
- 开头：表水自证（"我是好人，我分析一下局势"）
- 中间：踩一个你认为可疑的玩家，给出理由
- 结尾：总结你的判断，呼吁大家投票
