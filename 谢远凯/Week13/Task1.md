DeepSeek-V3 在 DeepSeek-V2 基础上进行了全方位架构、训练机制、数据体系与性能优化升级：模型采用超大稀疏 MoE 架构，总参数扩容至 671B、层数提升至 61 层，扩充专家数量并优化单 Token 激活专家配置，在提升模型容量的同时保持推理计算高效；摒弃 V2 依赖辅助损失做专家负载均衡的方式，创新无辅助损失的负载均衡策略，通过可学习偏置动态调控专家分配，训练更稳定且规避正则化副作用；训练目标从单 Token 预测升级为多 Token 预测 MTP，既提升预训练数据效率与长序列建模能力，又可支持推测解码加速推理；预训练数据规模从 2T 扩充至 14.8T 高质量多语言语料，大幅提升数学、代码、科学类数据占比，严格过滤去重增强数据多样性与质量；训练层面引入 FP8 细粒度混合精度搭配自研 DualPipe 通信优化，显著节省显存、降低通信开销与训练成本；在后训练阶段引入 DeepSeek-R1 推理能力蒸馏，将长思维链、多步逻辑推理能力融入模型；同时推理调度与架构深度优化，生成速度达到 V2 的 3 倍，在 MMLU、数学、代码、专业基准等各项评测指标上均实现大幅跃升，综合通用能力、数理推理、代码生成及中文理解能力全面超越 DeepSeek-V2。
